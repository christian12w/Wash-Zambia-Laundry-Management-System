# 🚀 GitHub Deployment Guide

## 📋 Prerequisites
- GitHub account
- Git installed on your system
- Wash Zambia Laundry Management System files ready

## 🎯 Step-by-Step GitHub Upload

### 1. Create GitHub Repository
1. **Login to GitHub**: Visit https://github.com
2. **Create New Repository**: Click "+" → "New repository"
3. **Repository Details**:
   - Repository name: `wash-zambia-laundry`
   - Description: `🧺 Professional Laundry Management System for Zambian Businesses`
   - Visibility: `Public` (or `Private` for personal use)
   - **DO NOT** initialize with README, .gitignore, or license
4. **Click "Create repository"**

### 2. Connect Local Repository to GitHub
```bash
# Replace YOUR_USERNAME with your actual GitHub username
git remote add origin https://github.com/YOUR_USERNAME/wash-zambia-laundry.git
```

### 3. Push to GitHub
```bash
git push -u origin master
```

### 4. Verify Upload
1. **Visit your repository** on GitHub
2. **Check all files are present**:
   - ✅ wash-zambia-laundry.html (main application)
   - ✅ README.md (documentation)
   - ✅ LICENSE (MIT license)
   - ✅ .gitignore (excludes unnecessary files)
   - ✅ Wash Zambia log.jpg (logo)

## 🌐 GitHub Pages Deployment (Optional)

### Enable GitHub Pages
1. **Go to repository settings**
2. **Scroll to "Pages" section**
3. **Source**: Deploy from a branch
4. **Branch**: master
5. **Folder**: /(root)
6. **Click "Save"**

### Access Your Live Site
- **URL**: `https://YOUR_USERNAME.github.io/wash-zambia-laundry/`
- **Direct access**: `https://YOUR_USERNAME.github.io/wash-zambia-laundry/wash-zambia-laundry.html`
